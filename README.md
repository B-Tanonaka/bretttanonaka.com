## Getting Started

Installation
- Clone the repository
    ```
        git clone https://github.com/B-Tanonaka/bretttanonaka.com.git
    ```
- Install the dependencies
    ```
        npm install
    ```
- Compile and build the files
    ```
        npm run build
        npm run bundle
    ```
- Start the program
    ```
        npm start
    ```
